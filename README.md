# STM32步进电机开环控制系统

## 项目简介
本项目是基于STM32F103微控制器的**步进电机开环控制平台**，实现了电机方向控制、多档速度调节、启停控制及串口指令解析等功能，可作为工业自动化、智能小车等场景的基础控制模块。

## 核心功能实现
- **电机方向控制**：支持电机正转和反转切换
- **速度调节**：通过定时器分频实现多档速度控制，避免电机丢步
- **启停控制**：支持电机的启动、停止和软启停，减少机械冲击
- **串口通信**：通过UART接收上位机指令，实现远程控制
- **指令解析**：支持解析方向、速度、步数等控制指令，实现灵活控制

## 硬件平台
- 主控芯片：STM32F103C8T6（Cortex-M3内核，72MHz）
- 驱动模块：A4988步进电机驱动板
- 执行器：42型两相四线步进电机
- 电源：12V/2A直流电源

## 软件架构
- 开发环境：Keil MDK-ARM + STM32CubeMX（HAL库）
- 核心文件：
  - `Core/Src/main.c`：主函数、系统初始化、电机控制逻辑
  - `Core/Src/uart.c`：UART串口初始化、数据发送与接收
  - `Core/Src/tim.c`：定时器配置，用于生成步进脉冲
- 关键技术：
  - 定时器PWM/脉冲输出控制电机转速
  - 串口中断处理变长指令数据
  - 状态机思想实现电机运行状态管理

## 项目亮点
1.  **模块化设计**：代码分层清晰，电机控制、通信、初始化模块独立，便于维护和扩展
2.  **工程化实践**：包含完整的硬件连接图、调试记录和功能说明
3.  **可扩展性强**：预留了编码器接口和PID控制接口，为后续闭环控制升级做准备

## 后续优化方向
- 增加增量式编码器，实现PID位置/速度闭环控制
- 实现梯形/S型加减速算法，优化电机启停平稳性
- 设计带校验的自定义串口协议，提升通信可靠性
- 移植FreeRTOS，实现多任务调度（电机控制+通信+状态监控）
