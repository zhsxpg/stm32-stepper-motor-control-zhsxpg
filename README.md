# STM32步进电机开环控制系统

## 项目简介
本项目是基于STM32F103微控制器的**步进电机开环控制平台**，实现了电机方向控制、多档速度调节、启停控制及串口指令解析等功能，可作为工业自动化、智能小车等场景的基础控制模块。

## 核心功能实现
- **电机方向控制**：支持电机正转和反转切换
- **速度调节**：通过定时器 PWM 实现无级速度控制，运行稳定
- **启停控制**：支持电机立即停止，安全可靠
- **串口通信**：通过UART接收上位机指令，实现远程控制
- **指令解析**：自定义帧格式解析，支持方向、速度、圈数控制

## 硬件平台
- 主控芯片：STM32F103C8T6（Cortex‑M3内核，72MHz）
- 驱动模块：A4988 步进电机驱动板
- 执行器：42型两相四线步进电机
- 电源：12V/2A直流电源
- 指示灯：PA3、PA4、PA5、PA6 用于状态指示

## 软件架构
- 开发环境：Keil MDK-ARM + STM32CubeMX（HAL库）
- 核心文件：
  - `Core/Src/main.c`：主逻辑、指令循环
  - `Core/Src/uart.c`：串口初始化、收发
  - `Core/Src/tim.c`：定时器 PWM 与中断配置
- 关键技术：
  - 定时器 PWM 输出控制电机转速
  - 自定义帧头帧尾串口协议
  - 定时器中断实现指定圈数控制
  - GPIO状态指示系统运行

# 串口通信协议（与代码完全对应）

### 通信参数
- 波特率：9600
- 数据位：8
- 停止位：1
- 校验位：无
- 帧格式：`0xAA + 指令 + 参数 + 0x5A`

---

## 指令集说明

### 1. 指令 0x01：转速 + 方向控制
格式：`AA 01 方向 速度 5A`
- 方向：
  - `04` → 正转
  - `05` → 反转
- 速度：`00~FF`，数值越大速度越快

现象：
- 正转：PA3 闪一下
- 反转：PA4 闪一下
- 电机按设定速度持续运转

示例：
- 正转、速度 20 → `AA 01 04 14 5A`
- 反转、速度 30 → `AA 01 05 1E 5A`

---

### 2. 指令 0x02：停止电机
格式：`AA 02 5A`

功能：立即停止 PWM 输出，电机停转

现象：
- PA5 灯闪一下
- 速度恢复默认

---

### 3. 指令 0x03：按圈数转动
格式：`AA 03 圈数 5A`
- 圈数：`01~FF`
- 1 圈 = 100 个脉冲

现象：
- 电机转动指定圈数后自动停止
- 完成后 PA6 闪烁对应圈数（转几圈闪几次）

示例：
- 转 2 圈 → `AA 03 02 5A`
- 转 5 圈 → `AA 03 05 5A`

## 串口返回值
- 执行成功：返回 `success`
- 指令错误：返回 `error`
- 参数错误：返回 `param err` / `angle err`

## LED 状态说明
- PA3：正转指示
- PA4：反转指示
- PA5：停止指示
- PA6：指定圈数完成指示
